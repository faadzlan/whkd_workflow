* 2. File & Folder Management

** Where do downloaded files go? How often do you clean Downloads?
Downloaded files go to the Downloads folder. I clean them whenever I notice opening the Downloads directory ends up hanging file explorer and the task needs to be restarted: which is a nuisance.

** How do you organize files by project, date, or something else?
Alphabetically and then chronologically.

** Ever need to rename many files at once?
Yes! Especially after coming back from a field trip or site visit, I have a lot of images that needed renaming. I have found a workflow in which I ask Kimi to look at the images and suggest names depending on the content of the images, then output the naming scheme in an org file for me to review (there is a confidence meter in the org file indicating how sure Kimi is in the names. Those flagged NEED REVIEW will be reviewed, together with those of low confidence)

** Do you sync files between folders or to cloud storage manually?
I do need to sync files from the WSL side to the Windows side. I sometimes work on the WSL side of files because I need to use git and github. So, there is a bash script I aliased to run inotify, detecting if any changes are made and then rsyncing it over to the Windows side for GDrive syncing to the cloud. The actual files are on the WSL side, on the GDrive are only symlinks - I think.
